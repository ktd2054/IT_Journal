# Microsoft 365 Email Security & Threat Protection Simulation

This Project simulates how an IT support handles email security, investigates phishing activities, fixes email delivery issues, and configures defender policies.

## Objectives

- protecting organisation from phishing and malware
- configure all defender email protections
- troubleshoot email delivery
- perform phishing simulation and analysis
- report findings

# Tools

- M365 Defender
- Exchange Admin Center
- Message Trace Tool
- Security and Compliance center
