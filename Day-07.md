# Hands-On: Microsoft 365 Business 

Getting Started with 1-month free trial.

Purpose: The purpose of exploring the Microsoft 365 Admin Center is to learn how a real IT Support / System Administrator manages users, security, devices, and company data inside a Microsoft cloud environment. When companies move to Microsoft 365, they rely on the Admin Center to run daily operations.

#### Task-1 Creating domain and registering users

- In menu ; users -> Active Users
- Click Add User and fill the details
- Check Auto-generate password
- Set license to M365 Business Premium

#### Task-2: Resetting Password

- Go to Users -> Active Users
- Select the User
- Click Reset Password from the Top Menu
- Uncheck Auto-generate
- Type password and click Reset Password Button

<img width="1912" height="796" alt="image" src="https://github.com/user-attachments/assets/2ba5b4bf-58c6-4803-8b41-c839bb9ededb" />

#### Task-3: Adding / Removing License

- Go to Users -> Active Users
- Select the user
- Click Manage Product Licenses from top menu
- Click the check box to add/remove license
- Click Save Changes Button
